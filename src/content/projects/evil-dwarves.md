Evil Dwarves is a top-down 3d shooter with fun ricochet mechanics that I developed as a solo for the [Bullet Hell Game Jam](https://itch.io/jam/bullet-hell-v) using Unreal Engine 5 (Blueprints) and Blender. The game features several enemy AI types with distinct behaviors.
